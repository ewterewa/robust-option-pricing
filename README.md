Robust Option Pricing Dashboard (ROPD)

Интеграция робастного проектирования и теории реальных опционов

*Автор:* Евтерева М.Д. (МОУ «СОШ №2» г. Всеволожска)  
*Научный руководитель:* Воробьёв Даниил Дмитриевич (Университет ИТМО)

Описание проекта
Интерактивный дашборд, реализующий трехэтапную модель оценки инновационных НИОКР:
1. Расчет индекса робастности (R)
2. Трансляция технического риска в финансовую волатильность
3. Оценка реального опциона по формуле Блэка-Шоулза
